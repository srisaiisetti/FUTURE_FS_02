🚀 Client Lead Management System (Mini CRM)

📖 Overview

A full-stack Mini CRM application that helps businesses manage customer leads from website contact forms. Admins can view leads, update their status, and maintain follow-up notes through a secure dashboard.

✨ Features

- Add and manage leads
- View all leads in a dashboard
- Update lead status (New, Contacted, Converted)
- Add follow-up notes
- Secure admin login
- Search and filter leads (optional)

🛠️ Tech Stack

- Frontend: React.js, HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- Tools: VS Code, Git, GitHub


🎯 Learning Outcomes

- CRUD operations
- REST API development
- Frontend-backend integration
- Database management
- Real-world CRM workflow
